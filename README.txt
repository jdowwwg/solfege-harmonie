Étapes pour la mise en ligne (Netlify Drop, le plus simple) :

  1. Allez sur https://app.netlify.com/drop
  2. Glissez-déposez le DOSSIER entier "solfege-site" (pas les fichiers
     séparément) — Netlify détectera automatiquement index.html.
  3. Vous obtenez une URL temporaire (ex. https://random-name.netlify.app).
     Créez un compte gratuit pour conserver le site et personnaliser
     le sous-domaine.

  4. Une fois votre URL définitive connue, ouvrez index.html dans un
     éditeur de texte et remplacez TOUTES les occurrences de
     "VOTRE-DOMAINE.example" par votre vrai domaine (ex. solfege.netlify.app).
     Cela active correctement les vignettes de partage et le SEO.
